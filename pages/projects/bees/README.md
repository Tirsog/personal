# UK Beekeeper's Calendar

A comprehensive seasonal guide for beekeepers in the UK, tracking key tasks, colony state, and available forage throughout the year.

## Overview

This project provides an interactive calendar showing:
- **Bee Forage Flowers**: What plants are blooming each month
- **Beekeeper's Work**: Monthly tasks and responsibilities
- **State of Colony**: How the bee colony behaves each month
- **Local Bee Forage**: Available nectar and pollen sources

## Technical Implementation

- Built with Next.js and React
- CSS Modules for styling
- Interactive tabs for switching between flower calendar and beekeeper's calendar
- Static data arrays (no API calls)
- Fully responsive design

## Data Source

Based on the British Beekeeping Association's "A YEAR'S WORK" guide (BBKA Revision Topic 1.14).

## Features

- Monthly breakdown of bee-friendly flowers
- Detailed beekeeper tasks for each month
- Colony lifecycle information
- Forage availability by season
- Notes on critical periods (swarming season, June Gap, etc.)
